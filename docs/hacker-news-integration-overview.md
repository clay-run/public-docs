# Hacker News integration overview

Discover tech news, trends, and discussions in real-time

## **Hacker News Overview**

The Hacker News integration in Clay allows users to pull key data from posts and user profiles on Hacker News.

## **Available Actions with the Hacker News Integration**

### **`Action` Find Post Details**

Use this action to retrieve key data from a specific Hacker News post by providing its URL.

**Setup Inputs**

-   **Hacker News Post URL**: Enter the URL of the Hacker News post (e.g., [https://news.ycombinator.com/item?id=23700043](https://news.ycombinator.com/item?id=23700043)).

### **`Action` Find User Details**

Use this action to fetch comprehensive details about a Hacker News user based on their profile URL or username.

**Setup Inputs**

-   **Username or URL**: Enter the username or profile URL of the Hacker News user (e.g., [https://news.ycombinator.com/user?id=Alex3917](https://news.ycombinator.com/user?id=Alex3917) or Alex3917).
